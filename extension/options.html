GitHub source (optional, recommended for live updates)

When the local bridge isn't running, the extension can pull live pipeline data + tailored PDFs straight from your private repo. Paste a fine-grained Personal Access Token below; it's stored only in this browser.

How to create a token

  1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Repository access: "Only select repositories" → pick career-ops (or whichever repo holds your pipeline).
  3. Permissions → Repository permissions → Contents: Read-only. Nothing else needed.
  4. Set an expiration (90 days is fine; you'll just regenerate).
  5. Generate, copy the token, paste it above, and click Save and test.

The extension never sends the token anywhere except api.github.com and raw.githubusercontent.com. Stored in chrome.storage.local on this browser only.
